# Htmlucinate

Rendering web pages is hard. Why not let an LLM do it instead?

## How it works

0. Add your `OPENAI_API_KEY` to your env
1. `just run`
2. Open [localhost:3000](http://localhost:3000)
3. Type a URL or query into the bar and hit enter
4. Wait an unreasonably long amount of time
5. Click on stuff in the image
